# Navigating the App Menus

From Home (on startup)
1. Clicker - Counts how many times you click it
2. Calculator - Performs calculations from any base, to any base
3. Convertor - converts bases of a number
4. Differentiation - does differentation of a number (assuming the number is a polynomial, base being the term)
5. Expander - Expansion of a double-sided base
6. Fabricator - Fabricates a base based on a polynomial expression (function)

This is the same as in the sidemenu (AppShell), except the clicker is replaced with a button to go to the homescreen.
