# Basal Library
The code library for my Basal project.

The Python file is meant for sets of data to be put on a graph, and was programmed at 2AM.
The C# file is meant for encryption and hashing.
The C++ version is for ROCm and should be able to run in any operating system, but I have not tried yet, and is based on the python version.

    The directories 'list_syntax' and 'word_syntax' are libraries that use different syntaxes for their calculations, as well as input and output.

There is a directory 'other_code' for code related to Basal, but is not the core module.
