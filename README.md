# Basal Library
The library of files for my concept of a branch of mathematics (basal).

The code files are for proving (computationally) that the system is correct.

The directory \Documentation\ is for how a person would use the functionality of basal alongside normal mathematics - not finished yet. Basal should be used alongside normal maths, not its own separate system, as it works with the already existing number system.

The Python file is meant for sets of data to be put on a graph, and was programmed at 2AM.
The C# file is meant for cryptography.
The C++ version is for areas where other languages are not as good: ROCm, microcontrollers, cross-system desktop development; it is based on the python version.

    The directories 'list_syntax' and 'word_syntax' are libraries that use different syntaxes for their calculations, as well as input and output.

There is a directory 'other_code' for code related to Basal, but is not the core module.
