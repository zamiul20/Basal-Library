# Using the Python files

Constructor - `__init__()`: Creates an object of the class Basal, with the parameters that are given in the brackets

Functions (using ebass as the object of class Basal), not all functions are in both files:
1. ebass.Conv : Returns the value (float, could also be int) to a number of base ebass
2. ebass.Donv : Returns the number (string or list) from a base of ebass to a value (int or float)
3. Basal.tobasten : Returns the value (float or int) of a number (list or string) from a base and given velocity
4. Basal.notobasten : Returns the number (list or string) of a value (int or float) from a given base, accuracy and velocity
5. Basal.polariser : Returns the value (int or float) of a given number (string or list) from a given base, velocity and polarity
6. Basal.depolaris : Returns the depolarised variant of a number (string or list) from given base, velocity, accuracy, polarity
7. Basal.unechor : Derippllificates the input number from a given ripple and echo
8. Basal.echor : Rippllificates the input number from a given ripple and echo
9. ebass.ct_disp : Creates a string to display the base's attributes
10. ebass.disintegrate : Returns the derivitive of the given polynomial (basal notation)
11. ebass.integrate : Returns the integral (of one point) of a given polynomial (basal notation) and constant
12. Basal.expor : Returns the value of exponentiation (kinda)
13. Basal.Cn : Returns the unicode position of a character relative to '0'
14. Basal.Vn : Returns the character of a value (48 is added)
15. Basal.shift : Returns a list shifted be a given value of positions (int)
