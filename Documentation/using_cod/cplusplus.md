# Using the C++ files

The Basal.cpp file in /array_syntax/ is under development, and may not work as intended

Constructor - `Basal()` - makes an object with default parameters, entered parameters from number to value or value to number

Functions (using ebass as the object of class Basal), not all functions are in both files, read parameters of function to know what to input:
1. ebass.expor() : exponentiation of a number to a power (mostly)
2. ebass.tobasten() : returns the value of a number
3. ebass.notobasten() : returns the number of a value
4. ebass.polaris() : returns the value of a depolarised number
5. ebass.depolaris(): returns the depolarised variant of a number
6. ebass.unechor() : returns the regular version of a echo-ed number
7. ebass.echor() : returns the echo-ed version of a regular number
8. ebass.logbas() : returns the logarithm of the first parameter from the second parameter
9. ebass.Convert() : sets the number of a value of the object the function was called from in the variable `number`
10. ebass.Deconvert() : sets the value of a number of the object the function was called from in the variable `num_value`
