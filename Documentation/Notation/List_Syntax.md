# List Syntax

List syntax is a method of arranging a number where its digits' values are separated by commas (or any other separating character, define in a key or by 'Let').
This is also interchangable to word syntax, and you can use both at any time, or just one, as it does not change the number's value or any attribute.

The core concept is to separate every digit and put it in an array / list.

This makes bases larger than 10 easier to comprehend and use, as after using numerical digits, we have to move to another characterset, being Unicode, ASCII or another alphabet.

An example using Hexadecimal - <16T:
```
<10T 178 = <16T B2 = <16T 11, 2
```

Another reason to use this syntax is for use with the basal calculator apps, updated code, simplifying polynomials, and to use variables and decimals as digits.
