Polarities of bases in basal.

1. Regular
2. Negative
3. Double Negative

Regular:

The base is exponentiated as-is, and every index's value is given from the equation ( (b)^ i ) where [b] is the base's value and [i] is the index.

Negative:

As a result, every other index from the position '1' is negative, and every other position is positive.

Double Negative:

As a result, every other index from the position '0' is negative, and every other position is positive.
