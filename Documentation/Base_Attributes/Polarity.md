# Polarities of bases

1. Regular
2. Negative
3. Double Negative

Regular:

The base is exponentiated as-is, and every index's value is given from the equation ( (b)^ (i * v) ) where [b] is the base's value, [i] is the index and [v] is the velocity.

Negative:

Every index's value is calculated from the equation : ( (-1)^ |i| x (b)^ (i x v) ) where [b] is the base's value, [i] is the index and [v] is the velocity.<p>
As a result, every other index from the position '1' is negative, and every other position is positive.

Double Negative:

Every index's value is calculated from the equation : ( (-1)^ (|i| + 1) x (b)^ (i x v) ) where [b] is the base's value, [i] is the index and [v] is the velocity.<p>
As a result, every other index from the position '0' is negative, and every other position is positive.

Machined Negative:

Every index's polarity (- / +) is given if the index is a term in a given equation (nth-term), with every other index being the opposite polarity.
