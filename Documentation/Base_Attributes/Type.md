Types of bases in basal.

1. Regular
2. Fractal  : Model 1,
            : Model 2
3. ONE

Regular Type:

An integer base value where [b] is the base and satisfies the equations (b > 1) and (⌊b⌋ = b) .

Fractal Types:

Model 1 : 

A non-integer base value where [b] is the base and satisfies the equations (b > 1) and (⌊b⌋ ≠ b) .

Model 2 :

A non-integer base value where [b] is the base and satisfies the equation (0 < b < 1) .

ONE:

Base is 1. Note: best used with discretion (See Format.md).
